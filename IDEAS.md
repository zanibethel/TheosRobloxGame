# TheosRobloxGame Ideas

## Game Concept

A 1–6 player co-op horror escape game.

Players start in a dark room and must search for items to escape to the next room while hiding from scary entities. If an entity catches a player, they die and have to start over.

## Core Gameplay Loop

1. Spawn in a dark room.
2. Search the room for helpful items.
3. Avoid or hide from entities.
4. Use items to unlock the next room.
5. Escape to the next room.
6. Repeat with harder rooms and scarier entities.

## Searchable Spots

- Closets
- Under beds
- Behind doors
- Drawers
- Cabinets
- Boxes
- Vents
- Wall panels
- Trash piles
- Paintings
- Desks

## Items

### Escape Items

- Key
- Passcode note
- Fuse
- Crowbar
- Door handle
- Map piece

### Survival Items

- Flashlight
- Battery
- Force field
- Medkit
- Noise maker

### Trick Items

- Fake key
- Cursed note
- Entity hiding spot
- Item that makes noise
- Trap drawer

## Item Rules

- Weapons do not kill entities.
- Flashlights can stun entities for a short time.
- Force fields protect a player once.
- Batteries recharge flashlights.
- Noise makers distract entities.

## Co-op Ideas

Some doors require two or more players.

Examples:

- Two-player heavy door
- Two pressure plates
- One player holds a lever while another player opens the door
- Code split between two players
- One player distracts an entity while another searches

## Entities

- Watcher: moves when players are not looking
- Listener: hunts players who make noise
- Shadow: stronger in dark rooms
- Mimic: pretends to be a player
- Collector: steals items before attacking

## First Build Goal

Build one simple playable room:

- Dark spawn room
- One hidden key
- One locked door
- Door opens when player has key
- Exit leads to next room
